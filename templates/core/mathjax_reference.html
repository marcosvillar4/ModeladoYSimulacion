Consejos rapidos

  • Use variables simples como x, y.
  • Para multiplicacion implicita: 2x, xy.
  • Para exponentes use ^ o llaves: x^{10}.
  • Para fracciones use \\frac{a}{b}.
  • Para raices use \\sqrt{x}.
